## 🚨 If Something Goes Wrong

### ❌ "Build failed"
**Check:** Render logs for specific error message  
**Usually:** Missing environment variable  
**Fix:** Verify all 5 env vars are set in Render dashboard

### ❌ "Connection refused - MongoDB"
**Check:** Is MongoDB IP whitelist updated?  
**Fix:** Add Render's IP or `0.0.0.0/0` in MongoDB Atlas Network Access

### ❌ "503 Service Unavailable"
**Check:** Health check failing  
**Reason:** App taking too long to start or MongoDB connection failing  
**Fix:** Check logs, verify all env vars set correctly

### ❌ "Takes 30+ seconds to deploy"
**Reason:** Normal on free tier (cold start)  
**Solution:** Upgrade to Starter plan ($7/month) for always-on
